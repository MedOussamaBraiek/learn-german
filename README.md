## ✨ Features

- **5 languages** — Deutsch, Français, Español, English, العربية
- **4 levels each** — A1 (Beginner) → B2 (Upper Intermediate)
- **Random quizzes** — 10 questions per round, shuffled every time
- **AI-generated questions** — infinite unique questions from vocabulary templates
- **Instant feedback** — correct/wrong with explanations after each answer
- **Detailed results** — score, star rating, category breakdown bars
- **Mistake review** — see exactly what you got wrong with corrections
- **🤖 AI Analysis** — personalized study tips based on your weak/strong areas
- **Multi-language UI** — switch interface between English, Arabic, French, German
- **RTL support** — Arabic UI renders right-to-left
- **Animations** — bounce, shake, slide, pop — fun Duolingo-like feel
- **Responsive** — works on mobile and desktop

## 🧑‍💻 Development

```bash
# install dependencies
npm install

# start dev server
npm run dev

# build for production
npm run build
```
